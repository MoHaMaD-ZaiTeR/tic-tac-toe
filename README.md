# tic-tac-toe
This project is a command-line implementation of the classic Tic-Tac-Toe game in Python, featuring a human player against an intelligent computer opponent.

## Features
- Interactive 2-player gameplay (User vs Computer)
- Rule-based AI that plays strategically
- Win detection for rows, columns, and diagonals
- Draw detection
- Input validation for a smooth user experience

## AI Logic
The computer uses a simple decision-making strategy:
- Plays winning moves when available
- Blocks the opponent’s winning moves
- Prioritizes center and corner positions
- Falls back to random valid moves when needed

## Technologies Used
- Python
- Basic algorithms
